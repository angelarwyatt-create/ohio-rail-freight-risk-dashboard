# ohio-rail-freight-risk-dashboard
Rail freight commodity dependency and network risk analysis using FAF data
This project analyzes inbound rail freight flows into Ohio using the Freight Analysis Framework (FAF 5.7.1) dataset from the U.S. Department of Transportation.

The dashboard evaluates strategic freight risks relevant to railroad network planning and operations, including:

• Coal dependency and exposure to energy market transitions  
• Commodity concentration risk within rail freight flows  
• Geographic origin-state dependency that may create operational disruption risks from weather or natural disasters  

The analysis is designed as an executive-style dashboard that highlights freight mix trends and potential vulnerabilities in the rail supply network serving Ohio.
Data Source
Freight Analysis Framework (FAF 5.7.1)
U.S. Department of Transportation – Bureau of Transportation Statistics
